## 🎯 설명

React 문법 및 Router와 scss, TMDB API를 활용한 영화 사이트

## 🚀 주요 스택

- React
- SCSS
- API(TMDB)

## 🧩 구성

- Profile 페이지
  - 넷플릭스 로그인 후 프로필을 고르는 화면을 작성함.
- ManageProfile 페이지
  - 프로필 설정을 누를 시에 나타날 페이지
- UserProfile 페이지
  - 수정할 프로필을 누른 뒤에 나타나는 페이지
- Movie 페이지
  - 영화 안내 페이지
- MovieDetail 페이지
  - 영화 상세 페이지
